# Hero Images Directory

Please add your Brown Sugar Bakery hero images to this directory:

1. `hero-chicago-skyline.jpg` - Chicago skyline with Field Museum and Willis Tower
2. `hero-chicago-river.jpg` - Chicago River with drawbridges and Marina City
3. `hero-buckingham-fountain.jpg` - Buckingham Fountain with Chicago skyline

## Image Requirements:
- Format: JPG, PNG, or WebP
- Recommended size: 1920x1080px or larger
- Aspect ratio: 16:9 or similar landscape orientation
- File size: Optimized for web (under 500KB each recommended)

The carousel will automatically cycle through these images every 5 seconds.
